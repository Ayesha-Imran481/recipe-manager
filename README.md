# Recipe Management System

📌 Project Overview
This is a PHP & MySQL based CRUD application to manage recipes, categories, and ingredients.

---

## How to Run (Steps)

### 1. Install Required Software
- XAMPP (Apache + MySQL)

### 2. Setup Project
Clone or download the repository into htdocs:

```bash
cd C:\xampp\htdocs\

3. Start Server :

Open XAMPP and start:
> Apache
> MySQL

4.Database Setup:
> Open Browser and enter: 
http://localhost/phpmyadmin
> Create a database:
   recipe_manager
> Import the provided SQL file

5. Run Project
Open browser and go to:
http://localhost/your-project-folder/
